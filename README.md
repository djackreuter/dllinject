# dllinject

```
DLL Inject
0xbu117
Takes a remote or local DLL and injects it into a target process.

USAGE:
    dllinject.exe [OPTIONS] --process <process>

OPTIONS:
    -h, --help                 Print help information
    -l, --local <local>        Path to local DLL.
    -p, --process <process>    Process name to inject into. e.g., mstsc.exe
    -u, --url <url>            URL to remote dll file.
    ```
